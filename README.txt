== Description ==
This plugin empower user to filter the existing order by the different fields.
Just simply check what type of filter user want to filter with and the filter on order detail page.


== Installation ==

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on the Setting option of plugin after activation OR Navigate to ** WooCommerce > Settings > Pay Filter.
4. Check the fields.
5. Save Changes.
6. In woocommerce order click on Additional Filter

== Frequently Asked Questions ==

= How many filter are their. =

Their are plenty like customer name, sku, payment method, user email, order status, billing country etc.
